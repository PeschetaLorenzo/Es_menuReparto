# Es menu 

Sito per l'ordinazione di hotdog per una cena di autofinanziamento di un gruppo scout.
Formato da due pagine: 
1. Pagina "utente": da qui è possibile effetuare l'ordine, selezionando gli ingredienti che desidera all'interno dell'hotdog e quale contorno.
2. Pagina cucina: visualizza tutti gli ordini non ancora cmpletati, è possibile selezionare gli ordini per completarli
